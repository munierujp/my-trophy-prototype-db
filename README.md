# my-trophy-prototype-db
Database of My Trophy Prototype

## Prepare
### Create .env file
```sh
$ cp .env.template .env
```

Input values to `.env` file.

## Usage
```sh
# run
$ docker-compose up -d
# stop
$ docker-compose stop
# start
$ docker-compose start
# remove
$ docker-compose down
````
